Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

↑Obeserve que os espaços das letras não são padronizados, cada um ocupa um espaço diferente, dificultando assim a leitura do item.↑

Agora observe o mesmo comando document.getElementById('teste') é escrito em linguagem JavaScript, com a tag code.

↑Obeserve que os espaços das letras agora são padronizados, e cada um ocupa o mesmo tamanho, facilitando assim a leitura do item.↑

CODE

No exemplo abaixo utilizamos a tag code, para escrever um código python, e o pensamento mais lógico é achar que ele vai apresentar a identação feita, porém olha como fica:

num = int(input('Digite um numero')) if num % 2 == 0: print(f'0 número {num} é PAR') else: print(f'0 número {num} é IMPAR) print('Fim do programa')

Para que o code possa vir com a identação, é interessante envelopar ele com a tag pre, olha como fica ↓

        
    num = int(input('Digite um numero'))
    if num % 2 == 0:
        print(f'0 número {num} é PAR')
    else:
        print(f'0 número {num} é IMPAR)
    print('Fim do programa')
